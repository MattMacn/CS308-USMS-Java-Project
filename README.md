## Overview
**USMS** is a university management system built as part of a team project for the CS308 course.  
The system is designed to streamline university administrative tasks, including managing courses, students, and faculty information.  

This project was developed collaboratively by a team of students, with clearly defined contributions for each member.

## My Contributions
In this project, I was primarily responsible for:
- Designing and implementing core backend functionality in Java
- Creating database schemas and SQL scripts for data storage
- Writing unit tests to validate key modules
- Integrating components into a functional system
